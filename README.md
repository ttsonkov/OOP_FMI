Лекциите за курса Обектно-ориентирано програмиране във ФМИ. Идеята на курса е да въведе студентите в основите на ООП, базирано на езика С++. Набляга се на модерни концепции от езика (включително С++ 23) както и сравнения с езика Java на концептуално ниво.
От първата до последна лекция се разглежда една предметна област (class Student), която се надгражда с всяка лекция.
